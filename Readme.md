# Data Structures and Algorithms Menu-Driven Program (C++)

## 📌 Overview

This project is a **Menu-Driven C++ Program** that demonstrates the implementation of fundamental **Data Structures and Algorithms (DSA)** concepts.

The program includes:

* Linked List Operations
* Merge Sort
* Quick Sort
* Binary Search
* Array Management
* Interactive Menu System

It is designed for students and beginners who want to understand the practical implementation of DSA concepts in C++.

---

## 🚀 Features

### 1. Linked List

* Insert nodes at the end of the linked list
* Display all linked list elements

### 2. Merge Sort

* Recursive Divide and Conquer sorting algorithm
* Efficient for large datasets
* Time Complexity: **O(n log n)**

### 3. Quick Sort

* Pivot-based sorting algorithm
* Fast average performance
* Time Complexity:

  * Best/Average: **O(n log n)**
  * Worst: **O(n²)**

### 4. Binary Search

* Searches an element in a sorted array
* Time Complexity: **O(log n)**

### 5. Array Operations

* Enter array elements
* Display array contents
* Apply sorting algorithms
* Perform searching operations
